## 結構

```
.
├── app.py           # 主程式：自動偵測韓國今天日期，跳到當日 Day
├── utils.py         # 渲染：彩色 G/N/T 按鈕、主行程卡片、備案 expander
├── places.py        # 資料中心：所有 60+ 個地點 (集中管理)
├── day1.py ~ day5.py
└── requirements.txt
```

## 三顆按鈕

每個地點都有：
- **G** (Google Maps，藍色) — 跨平台
- **N** (NAVER 地圖，綠色) — 需安裝네이버지도 App，韓國當地最準
- **T** (Kakao T，黃色) — 一鍵叫車，開啟後輸入目的地

## 設計

- 每個主行程下方有 **「附近備案」expander**：同區其他選擇 (按吃/逛/景點分類)
- 每天底下有 **「飯店附近」expander**：弘대區常駐清單
- 頂部 **Day Tab 橫向滑動**，自動偵測韓國今天日期跳到當日
